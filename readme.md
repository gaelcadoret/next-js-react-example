# Next JS - React project
- Simple ***asynchronous state update*** (with native javascript ***fetch*** method) without redux
- Toggle theme feature
- React hooks : ***useState***
- ***Statefull*** and ***stateless*** component
- Sass version
- Async / Await version

## Install dependancies
`yarn`

## Run localhost:3000
`yarn dev`
